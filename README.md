# HEYA!

Basically what this is is an attempt to make as complex of a 3D engine as possible.  
My first attempt at making 3D was in BG3D but it was really lacking.

### Things I want to implement to this engine:
- Full OOB support (slopes, rotating objects, etc)
- Good shaders
- Non-euclidean geometry (most importantly: portals)
- A better room editor (with UI instead of just commands)
- An import/export format for levels
- Engine and editor fused into one single executable for showcase
- Export to a package for future use (?)
